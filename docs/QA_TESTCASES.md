## 테스트 실행 절차

### 사전 준비
```
1. https://aha-five.vercel.app 접속
2. 브라우저 개발자 도구 열기 (F12)
3. Console 탭 오류 없는지 확인
4. Application → Local Storage 초기화 여부 결정
   - 클린 테스트: 초기화 후 시작
   - 영속성 테스트(TC-12): 초기화 없이 시작
5. 데모 계정 준비: demo@aha.com / demo1234
6. admin 계정 준비 (TC-11용): admin@aha.com / admin1234
```

### 권장 실행 순서
```
1. TC-01 (레이아웃)     → 기본 UI 동작 확인
2. TC-02 (인증)         → 로그인 선행 필요
3. TC-03 (메뉴 렌더링)  → 17개 메뉴 순차 클릭
4. TC-05 (게시판)       → 로그인 상태에서 게시 / 댓글 / 좋아요
5. TC-12 (영속성)       → 새로고침 후 데이터 유지 확인
6. TC-06 (이모지 반응)  → 게시글 상세에서 반응
7. TC-04 (크롤링)       → admin 크롤링 선행 후 확인
8. TC-07 (검색)         → 게시글 + 크롤링 통합 검색
9. TC-08 (소셜)         → 팔로우 / 북마크 / 마이페이지
10. TC-09 (라이브)       → 라이브 이슈 + 채팅
11. TC-10 (숏폼)         → 데스크톱 + 모바일 환경
12. TC-11 (관리자 연동)  → admin-vert-psi.vercel.app 병행 필요
13. TC-13 (성능)         → DevTools Network / Memory 패널
14. TC-14 (접근성)       → Lighthouse 또는 수동 검사
```
